# Space Invaders – COMP.2800 Assignment #2

## Overview

This project is a browser-based implementation of the classic **Space Invaders** game created using **HTML5 Canvas, CSS, and JavaScript**.
The objective of the game is to control a spaceship, destroy incoming enemies, and survive as long as possible while progressing through increasing difficulty levels.

This project was developed as part of **COMP.2800 – Software Development**.

---

## Features

* Player-controlled spaceship
* Enemy formations that move horizontally and descend
* Player shooting system
* Enemy shooting mechanics
* Score tracking
* Lives system
* Level progression
* Pause functionality
* Game over and restart system
* Animated star background

---

## Controls

| Key              | Action                        |
| ---------------- | ----------------------------- |
| ← / → Arrow Keys | Move spaceship left and right |
| A / D            | Alternative movement controls |
| Space            | Shoot                         |
| Enter            | Start / Restart game          |
| P                | Pause / Resume                |

---

## How to Run the Game

1. Download or clone this repository.
2. Open `index.html` in any web browser.
